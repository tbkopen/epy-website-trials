## Hi, I'm [Your Name]

I write about mathematics, computing, and the places where they intersect. Elastropy started because I kept finding that the clearest explanations — the ones that actually make rigorous ideas *click* — don't exist yet. So I write them.

## What you'll find here

The blog covers topics like:

- Linear algebra and its computational geometry
- Probability theory and statistical learning
- Algorithms with provable guarantees
- Mathematical foundations of machine learning

Each post aims to be the resource I wish had existed when I was learning the topic. That means proofs when they add intuition, concrete code when it helps, and no hand-waving.
