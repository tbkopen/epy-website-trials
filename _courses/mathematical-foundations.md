## What you'll learn

This course builds the mathematical infrastructure modern machine learning sits on — starting from first principles and ending at the doorstep of deep learning.

### Module 1 — Vector spaces and linear maps

We begin where linear algebra begins: with the abstract definition of a vector space, not just arrows in $\mathbb{R}^n$. You'll learn why abstraction here is not pedantry — it's what makes the same ideas work for polynomials, functions, and matrices alike.

### Module 2 — The geometry of matrices

Eigendecomposition, singular value decomposition, and the four fundamental subspaces. Proof of the Spectral Theorem. Applications: PCA, least squares, low-rank approximation.

### Module 3 — Probability from measure theory

Sigma-algebras, probability measures, random variables — done properly. Conditional expectation as a projection. Law of large numbers and central limit theorem with proofs.

### Module 4 — Statistical inference

Maximum likelihood estimation. Bias-variance tradeoff. Concentration inequalities (Markov, Chebyshev, Hoeffding). The PAC learning framework.

### Module 5 — Optimisation

Convexity and why it matters. Gradient descent convergence proofs. Stochastic gradient descent. Introduction to constrained optimisation (KKT conditions).

### Modules 6–12

Neural networks as function approximators. Backpropagation as reverse-mode automatic differentiation. Generalisation bounds. And more.

## Who this is for

You should be comfortable with:

- Calculus (derivatives, chain rule, partial derivatives)
- Basic programming in Python
- High-school level linear algebra (matrices, vectors, dot products)

You do **not** need prior experience with proofs or abstract mathematics — the course builds that skill explicitly.

## Why rigour?

Most ML courses teach you to use tools. This course teaches you to *understand* them — so that when a method fails, you know why, and when a new method appears, you can evaluate it on first principles.

$$
\text{Understanding} = \text{Intuition} + \text{Rigour}
$$

Both halves matter. This course delivers both.
